# TubeVault

TubeVault is a curated video collection platform powered by AI.

## Features

- **AI Analysis**: Automatically tag and categorize videos.
- **Secure Storage**: Enterprise-grade vault for your personal archive.
- **Bento Grid Layout**: Modern, glassmorphic UI inspired by Velion AI.
- **Supabase Integration**: Scaleable backend.

## Tech Stack

- React + Vite
- Tailwind CSS
- Supabase
- Vercel

## Setup

1. Clone the repo.
2. `npm install`
3. Copy `.env.example` to `.env` and add Supabase keys.
4. `npm run dev`
